== Description ==

A generic items / child items custom post type plugin. Intended to be used as a base for more custom use cases.

Includes Metabox Fields for:
- Text
- Textarea
- Checkbox
- Date
- Image

== Installation ==

1. Upload `custom-boilerplate` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
